Le plugin skype permet de piloter SKYPE à l'aide de Sarah.

Pour cela, lors du lancement de WSRNode, le plugin SKYPE va s'initialiser et rentrer en contact avec l'application SKYPE.
Vous n'etes pas forcément obliger d'avoir une session SKYPE déjà ouverte pour que le plugin fonctionne.
Néanmoins:
Si une session SKYPE est déja ouverte vous pouvez directement interagir via Sarah.

Par contre si vous voulez que Sarah gère vos connexion(s) SKYPE, il faudra fixer correctement les paramètres du plugin via l'interface web.
Les variables concernées sont notamment:
  • "Skype_path", le chemin où se situe l'application SKYPE (souvent "C:\Program Files\Skype")

  • Vous devrez également configurer les triplet NameX, UserX, PassX (X variant de 1 à 3). Ces triplets permettent de paramètrer jusqu'à 3 comptes SKYPE différents et administrable par Sarah.

    • NameX est l'intitulé du compte que Sarah peut reconnaitre vocalement,

    • UserX est le login du compte Skype associé,

    • PassX est le password du compte Skype associé.

    Une fois les paramètres saisies, veuillez relancer WSRNode pour que les nouveaux paramètres soient pris en compte.
    Vous pourrez alors demander à Sarah de se connecter à votre compte SKYPE en lui lancant: "Sarah, ouvre la session 'xxxxx'.

    Il est probable que lors de la première connexion du plugin sur un compte de SKYPE, SKYPE vous avertisse de cela et vous demande si "wscript.exe" est autorisé à agir sur lui. Confirmer que vous l'autorisez.

    Pour que Sarah prenne en compte votre liste d'amis, vous devez également demander à Sarah d'importer vos amis dans la configuration du plugin. Pour cela, lors de la première utilisation (ou à chaque fois que vous désirez mettre à jour la liste de vos amis dans Sarah) dites:
    "Sarah, Mets à jour ma liste d'amis"
    Seuls les amis qui feront partis de la liste "Favorites" sur Skype seront ajouté, les autres seront ignorés. Veuillez donc créer la liste "Favorites" qui n'existe pas par défaut dans SKYPE et mettez tous les amis que vous voulez pouvoir joindre grace à Sarah dans cette liste

    Pour une utilisation intégré de SARAH et de SKYPE, j'ai de mon coté supprimer la sonnerie SKYPE qui est naturellement remplacé par la voix de SARAH et la présentation de l'appelant. Sinon si on a la sonnerie + SARAH qui parle ca fait un certain brouhaha.
